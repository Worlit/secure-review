# Security Policy

## Supported Versions

At the moment, only the latest version of the project is supported.

| Version | Supported |
| ------- | --------- |
| main    | ✔️        |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via:

- GitHub Issues (with the `security` label), or
- GitHub Security Advisories (private disclosure)

Please do not publish vulnerability details publicly before it is reviewed.
